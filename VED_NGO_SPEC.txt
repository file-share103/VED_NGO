TECHNICAL SPECIFICATION FOR VED NGO WEBSITE

Tech Stack:
- HTML, CSS, JavaScript
- Common navigation bar on all pages
- use the exesting folder structure
- Fully responsive design for mobile, tablet, desktop

Pages to Create:
- index.html (Home)
- about.html (About)
- past-projects.html (Past Projects)
- future-projects.html (Future Projects)
- donate.html (Donate)
- volunteer.html (Volunteer Registration)
- contact.html (Contact Us)

GLOBAL UI REQUIREMENTS:
- Sticky navigation bar
- Floating donate button (red)
- Responsive layout with media queries
- Loading animation/page loader
- Smooth animations and transitions
- Clean typography and consistent color theme
- Reusable components: nav, footer, modals, cards

DONATE PAGE (donate.html):
- Left: Image
- Right: Donation form

Form Flow:
1. Enter amount
2. Show fields:
   - Name
   - Phone
   - Email
   - Donate Anonymously (skip name/phone)
3. Display account details with Copy buttons
4. Donor receipt confirmation message shown (manual sending later)
5. Recurring donation options: ₹100/mo, ₹500/mo, ₹1000/mo or once payment option
